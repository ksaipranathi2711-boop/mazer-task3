# Task 3 – Mazer Dashboard Customization

## Objective
This project customizes the Mazer admin dashboard to display data-driven cards for Total Users, Active Users, and New Signups. It demonstrates front-end integration with JSON data and custom styling.

## Technology Stack
- HTML
- CSS (Mazer default + custom styling)
- JavaScript (ES6)
- JSON (data source for dashboard)
- Bootstrap 5 (used in Mazer template)

## Folder Structure

mazer-task3/
 ├─ index.html
 ├─ assets/
 │    ├─ css/
 │    │    ├─ main.css       (Mazer CSS)
 │    │    └─ custom.css     (your custom styles)
 │    └─ js/
 │         ├─ main.js        (Mazer JS)
 │         └─ dashboard-data.js  (your JS for JSON data)
 └─ data/
      └─ data.json

## How to Run
1. Open the repository on GitHub.
2. Go to **Settings → Pages → Branch: main → / (root) → Save** to enable GitHub Pages.
3. Visit the provided GitHub Pages URL to see the live dashboard.

## Features
- Displays Total Users, Active Users, and New Signups from JSON data.
- Responsive design using Bootstrap 5.
- Custom styles applied via custom.css.
